Schitarea unei pagini web utilizand elemente Java-Script

    Utilizand elementele studiate pana in prezent sa incercam sa schitam o noua pagina web.Vom utiliza meniuri, functii in JavaScript, div-uri pentru a pozitiona diverse elemente precum si iframe pentru a realiza continut dinamic.

Utilizare iframe pentru realizarea paginilor dinamice

     In aplicatia de jos vom schita o pagina web dinamica, pagina care contine zonele importante cum ar fi:
  • antet
  • meniu
  • zona de continut dinamic
  • baza paginii
    Zonele antet, meniu, baza paginii au continut relativ constant. Zona de continut dinamic este zona dinamica in care se schimba continutul in functie de meniul selectat. Unele optiuni de meniu schinmba si continutul din baza paginii. Zona dinamica a fost realizata prin definirea unui div si invocarea metodei .innerHTML. Prin aceasta metoda se poate defini un nou continut al div-ului. Prin folosirea iframe se poate incarca un fisier html in cadrul div-ului. Din meniul principal sunt lansate diverse functii care invoca metoda .innerHTML pentru a schimba continutul diverselor div-uri.
