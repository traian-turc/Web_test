Web Server


  • Configurare Web Server

        LabVIEW dispune de facilitati pentru afisarea ecranelor "Front Panel" in pagini web. Pentru utilizarea facilitatilor WEB se fac o serie de configurari. Prima configurare se refera la "Web Server". Din meniul principal, se alege Tools-->Options-->Web Server si se obtine:



        Se valideaza optiunea "Enable Web Server". In acest moment, s-a validat functionarea Web Server-ului la adresa implicita: http://localhost:80 Folder-ul implicit pentru acest server fiind C:\Program Files\National Instruments\LabVIEW 2009\www. In acest folder vor fi plasate paginile WEB care vor afisa VI-uri. Pagina implicita fiind index.html respectiv : http://localhost:80/index.html
